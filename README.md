# Amazon Sales Analysis 

## Project Overview

The main objective of this project is to analyze Amazon sales data and extract meaningful insights about product categories, pricing, ratings, and discounts using Python-based data analysis techniques.

## Dataset
The Amazon sales dataset used in this project was collected from Kaggle.

The dataset contains information about:
- Product categories
- Actual price
- Discounted price
- Ratings
- Discount percentage
- Product details

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- BeautifulSoup
- Kaggle (Dataset Source)
- Google Colab

## Tasks Completed

### 1. Exploratory Data Analysis (EDA)
- Checked dataset structure and information
- Analyzed columns and data types
- Checked missing values and duplicates
- Performed statistical analysis

### 2. Data Visualization
Created visualizations to understand:
- Top product categories
- Actual price vs discount price relationship
- Rating distribution
- Highest discount products

### 3. Web Scraping
- Collected data from websites using Python libraries
- Converted extracted data into a structured format

## Key Insights
- Identified popular product categories
- Observed pricing and discount patterns
- Analyzed customer rating trends
- Understood product performance through visualization

This project helped me understand the complete data analytics workflow, from data collection and cleaning to exploration and visualization.
